# 마인크래프트 스탯 스크립트

마인크래프트 스탯 Skript입니다. 플레이어는 Shift + F 키 또는 명령어를 통해 GUI를 열고 획득한 포인트를 투자하여 캐릭터의 능력치를 강화할 수 있습니다.

## 파일 구성
- **`stat_a.sk`**: 이 스크립트의 메인 코드입니다. 필수적으로 적용해야 합니다.
- **`stat_b.sk`**: 선택적으로 사용하는 확장 코드입니다. 서버에 `SkQuery` 애드온이 적용되어 있을 때 자동 완성의 확장성을 더 해주며 해당 애드온이 존재해야만 정상적으로 작동합니다.

## 안내 사항 (필독)
- **상세한 주석 포함:** 코드 내부에 각 변수의 역할, 시스템적 한계, 예외 처리 이유 등이 **주석(`#`)으로 상세히 기록**되어 있습니다. 코드를 수정하거나 커스텀할 때 주석을 먼저 참고하시기 바랍니다.
- **어드민 명령어(`/stat set stat`) 주의점:** 
  - `on level change` 이벤트가 플레이어의 현재 레벨을 기준으로 보유 포인트(`stat`)와 사용 포인트(`usestat`)를 강제로 계산하여 동기화합니다.
  - 따라서 관리자 명령어로 `stat`(보유 포인트)을 직접 추가하더라도, **플레이어의 레벨이 변경(레벨 업/다운)되는 순간 시스템이 포인트를 재계산하면서 수동으로 추가했던 포인트가 증발**하게 됩니다. 스탯 포인트를 영구적으로 지급하려면 레벨 자체를 올려주거나 관련 수식을 수정해야 합니다.
- **버전 호환성:** 
  - 본 스크립트는 기본적으로 1.17 버전에 추가된 `amethyst shard(자수정 조각)` 아이템을 사용합니다.
  - 하위 버전(1.16 이하)에서 사용하려는 경우, 코드 내의 `amethyst shard` 아이템 이름을 해당 버전에 존재하는 다른 아이템(예: `diamond`, `emerald` 등)으로 변경하면 정상적으로 작동합니다.

## 스탯 계산법 및 수치 적용 공식

스탯 투자 횟수(use)에 따른 실제 수치(value) 및 인게임 적용 공식은 다음과 같이 계산됩니다.

### 1. 기본 능력치 스탯
- **공격력**
  - **계산 공식:** 현재 공격력 = 투자 횟수 * 0.28
  - **인게임 반영:** 일반 공격 데미지 성공 시, 기본 가한 데미지에 현재 공격력 수치가 그대로 누적 적용됩니다.
- **체력**
  - **계산 공식:** 최대 체력 = 10 + (투자 횟수 * 0.5)
  - **인게임 반영:** 스탯 창에서는 기본 체력(10)을 제외한 투자 수치만 표기되나, 내부적으로는 마인크래프트 최대 체력(max health) 속성에 합산 적용됩니다. (스탯 1번 투자 시 하트 반 칸 증가)
- **민첩 (이동 속도)**
  - **계산 공식:** 인게임 이동 속도 = (6 + (투자 횟수 * 0.3)) / 30
  - **인게임 반영:** 스탯 창에서는 기본 이속(6)을 제외한 투자 수치가 표기되며, 최종 적용 시에는 값의 왜곡을 막기 위해 30으로 나눈 실수값이 플레이어의 걷기 속도(walk speed)로 대입됩니다.

### 2. 전투 및 확률 스탯
- **방어력**
  - **계산 공식:** 현재 방어력 = 투자 횟수 * 0.1
  - **인게임 반영:** 피격 시 최종 데미지 계산 단계에서 방어력 수치만큼 즉시 차감(데미지 감쇄)됩니다. 방어력이 0 미만으로 내려가지 않도록 안전장치가 설계되어 있습니다.
- **크리티컬 확률**
  - **계산 공식:** 크리티컬 확률(%) = 투자 횟수 * 1
  - **인게임 반영:** 공격 성공 시 해당 확률 검사를 진행하며, 최대 수치는 **100%**로 제한됩니다.
- **크리티컬 데미지**
  - **계산 공식:** 최종 크리티컬 배수 = 1 + (투자 횟수 * 0.07)
  - **인게임 반영:** 크리티컬이 발동했을 때, 기존 가한 데미지와 스탯 공격력이 더해진 총합 데미지에 이 배수 수치가 곱해집니다.
- **흡혈량**
  - **계산 공식:** 최종 흡혈량 = 최종 가한 데미지 * (투자 횟수 * 0.03)
  - **인게임 반영:** 타격 성공 시 가한 최종 데미지에 비례하여 100% 확률로 플레이어의 체력을 회복(heal)시킵니다. (흡혈량 수치가 0보다 클 때만 발동 및 메세지 출력)

## 주요 기능

### 1. 스탯 및 GUI 시스템
- **스탯 창 열기 (`Shift + F` 또는 `/stat open`):** 3줄(27슬롯) 크기의 인벤토리 GUI를 통해 직관적으로 스탯을 투자합니다.
- **포인트 초기화:** `/stat pointreset` 명령어를 통해 투자한 스탯을 포인트 형태로 되돌릴 수 있으며, 관리자가 지정한 횟수 제한 내에서만 작동합니다.

### 2. 전투 및 예외 처리 시스템
- **전투 공식 반영:** 
  - 방어력 수치만큼 최종 데미지를 차감합니다. (방어력이 0 미만으로 내려가지 않도록 예외 처리 포함)
  - 크리티컬 확률에 따라 확률적으로 크리티컬 데미지 배수가 적용되며, 공격 시 고유 사운드가 재생됩니다.
  - 흡혈 스탯에 비례하여 가한 데미지의 일정 비율만큼 체력을 회복합니다.
- **공허 즉사 트리거:** 방어력이 극도로 높아 공허(Y 수치 -150 이하)에서 죽지 않는 현상을 방지하기 위해, Y 좌표가 -500 이하로 떨어질 시 강제 사망 시키는 메세지 및 즉사 트리거가 내장되어 있습니다.
- **부활 선택 창:** 사망 후 리스폰 시 관전자 모드로 전환되며, 경험치와 스탯을 유지할지 혹은 초기화하고 부활할지 선택할 수 있는 전용 GUI가 제공됩니다.

### 3. 사용자 설정 GUI (`/stat setting`)
- 흡혈 시 출력되는 힐량 메세지의 표시 여부를 On/Off 할 수 있습니다. (기본 On)
- 기본 공격 시 상대에게 가한 데미지 메세지의 표시 여부를 On/Off 할 수 있습니다. (기본 Off)

## 명령어 안내

### 일반 플레이어 명령어
- `/stat open`: 스탯 GUI 창을 오픈합니다.
- `/stat setting`: 개인 설정 GUI 창을 오픈합니다.
- `/stat pointreset`: 보유한 초기화 횟수를 1 차감하고 찍은 스탯을 포인트로 되돌립니다.

### 관리자(OP) 전용 명령어
- `/statfirstjoin`: 대상 플레이어의 스탯 및 설정을 강제로 초기화하고 초기 세팅을 진행합니다.
- `/stat view [<닉네임>]`: 본인 또는 특정 플레이어의 상세 스탯 수치 및 투자 횟수를 채팅창으로 확인합니다.
- `/stat reset [<닉네임>]`: 본인 또는 특정 플레이어의 스탯, 레벨, 경험치를 완전히 초기화합니다.
- `/stat set <스탯종류> <수치> [<닉네임>]`: 특정 플레이어의 특정 스탯 값을 강제로 설정합니다.
  - *적용 가능 스텟: attack, crit_ch, crit_dm, defense, health, lifesteal, speed, stat*
- `/stat rcount <add/set> <수치> [<닉네임>]`: 특정 플레이어의 포인트 초기화 가능 횟수를 수정하거나 추가합니다.
